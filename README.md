## Installation

```shell
# clone this repository
$ git clone https://github.com/leopard-work/leetcode

# go into project directory
$ cd MDShop

# install dependencies
$ npm install
```

### Run dev server

```shell
$ npm run start
```

Run dev server on [http://localhost:3000](http://localhost:3000)

### Build

```shell
$ npm run build
```
